# stock

stock项目分为以下几个模块：
1. 数据库部分，主要负责数据的定义和存储
2. 数据抓取平台，也就是所谓的爬虫
3. 计算平台，主要负责数据加工，对原始的数据计算，获得统计数据等
4. 特征抽取平台，主要负责特征的抽取，组合，归一化等
5. 模型的训练和验证，针对具体的目标，设计不同的模型，用于分类或回归；在部分数据上验证模型的效果，并得到指标（TPR，AUC）
